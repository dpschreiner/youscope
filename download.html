Current Release

YouScope R2016 Beta 2 (v2.0.0-beta.2)

The latest available version of YouScope is the Beta 2 of the R2016 release of YouScope. This version is stable, and only minor bug-fixes are expected until the final release. This release is only available for Windows XP and higher. It is available for 32 or 64bit systems, and as a combined version allowing quick switching between 32 and 64bit. We recommend the 64bit version.
A POSIX (Linux, MacOSX, ...) version is not provided due to a lack of demand. If you would like to have a POSIX release and could live with an only little tested version, please contact us via mail: info@youscope.org.

  • Windows 32-bit (XP or higher):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.2/YouScope_32bit_Installer.exe

  • Windows 64-bit (XP or higher):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.2/YouScope_64bit_Installer.exe

  • Combined Windows 32 and 64-bit (XP or higher):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.2/YouScope_3264bit_Installer.exe
