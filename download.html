Current Release

YouScope R2016 Beta 5 (v2.0.0-beta.4)

The latest available version of YouScope is the Beta 5 of the R2016 release of YouScope. This version is stable, and only minor bug-fixes are expected until the final release. This release is only available for Windows XP and higher. It is available for 32 or 64bit systems, and as a combined version allowing quick switching between 32 and 64bit. We recommend the 64bit version.
It is also possible to only download a stand-alone version of the measurement viewer, for example, to analyze measurement data on a personal PC not connected to the microscope. The measurement viewer is available both for Windows (32bit and 64bit combined) and POSIX (MacOSX, Linux) systems.
A POSIX (Linux, MacOSX, ...) version of YouScope is not provided due to a lack of demand. If you would like to have a POSIX release and could live with an only little tested version, please contact us via mail: info@youscope.org.

  • YouScope, Windows 32-bit (XP or higher):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.4/YouScope_32bit_Installer.exe

  • YouScope, Windows 64-bit (XP or higher):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.4/YouScope_64bit_Installer.exe

  • YouScope, Combined Windows 32 and 64-bit (XP or higher):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.4/YouScope_3264bit_Installer.exe

  • Measurement Viewer, Windows 32 or 64-bit:

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.4/measurement-viewer.zip

  • Measurement Viewer, POSIX (MacOSX, Linux, ...):

    https://github.com/langmo/youscope/releases/download/v2.0.0-beta.4/measurement-viewer.tar.gz

Old Releases

YouScope R2016 Beta 4 (v2.0.0-beta.4)

This version contains a serious bug concerning possible data loss when imaging microplates. In this bug, the image file names do not contain an identifier of the y-position of the well. Imaging microplates at different y-positions thus could lead to overwriting of files. We recommend either using an early version, or a later version of YouScope. The bug is only present in THIS beta release of YouScope.
