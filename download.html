Current Release

YouScope R2017 Beta 1 (v2.1.0-beta.1)

This is the first beta of the R2017 (v2.1.0) release of YouScope.
Since a lot of the code base considering how a measurement is run (new pause option) has changed, this version might not be as stable as the R2016 final release. Choose R2016 final release if stability is more important for you than the new features offered by this version.
New functionality/changes:

  • Measurements can now be paused and resumed. While paused, the microscope becomes unlocked and can be either controlled directly, or other measurements can be run. A paused measurement can, however, not be edited.
  • Measurements now possess metadata and a free form description. Both is exported as XML and HTML, replacing the old scope_settings.xml.
  • File names now allow for any special characters allowed by Windows, and not only a subset thereof.

    YouScope R2016 (v2.0.0)

    YouScope R2016 is the last stable release of YouScope. R2016 was extensively tested during several pre-releases, and demonstrated considerable robustness and user-friendliness.
    Most new features of R2016 were already available (and extensively tested) in the pre-releases. The highlights contain support for completely customizable non-regular microplates and -chips, easier configurable measurements, extended support for microfluidics, the possibility to edit measurements while running in a controlled way, and various new job types allowing to further customize imaging protocols.     YouScope R2016 Beta 6 (v2.0.0-beta.6)

    The latest available version of YouScope is the Beta 6 of the R2016 release of YouScope. Most of the changes in this version consider the support of non-rectangular microplates. Since a lot of the code base considering microplate measurements had to be changed, this version might not be as stable as the other beta releases. Choose Beta 5 or a later version if stability is most important for you.
    A POSIX (Linux, MacOSX, ...) version of YouScope is not provided due to a lack of demand. If you would like to have a POSIX release and could live with an only little tested version, please contact us via mail: info@youscope.org.
    New functionality/changes:

    • Support of custom, non-rectangular microplates. The microplates can have virtually any layout, including gaps, missing wells, different well sizes, and unregular positions.
    • Path optimizers were revisited: there are a few more options in the different optimizers, e.g. different metrics.
    • Complete refactoring of the microplate measurement: Position configuration should be significantly simpler, and configured positions can manually be changed afterwards.

    YouScope R2016 Beta 4 (v2.0.0-beta.4)

    This version contains a serious bug concerning possible data loss when imaging microplates. In this bug, the image file names do not contain an identifier of the y-position of the well. Imaging microplates at different y-positions thus could lead to overwriting of files. We recommend either using an early version, or a later version of YouScope. The bug is only present in THIS beta release of YouScope.
